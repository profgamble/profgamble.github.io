Introduction to Twine

PB692

Twine is an open source tool that allows people to create web-based narratives without needing to know HTML, CSS, or JavaScript. While those languages can be integrated into Twine, and Twine files download as .html, you do not need to have a programming background to work with Twine. It is great for making interactive stories, such as choose-your-own-adventures, and other non-linear works such as hypertext fiction.

In this activity, you will start exploring this tool by creating a story in Twine.

Activity Steps

  1. First, storyboard out a story that has three forks. For example, below is the storyboard for “Caturday.” Each blue circle represents a passage, which is what Twine calls a page, on which a reader will make a choice. Each arrow represents the passage where that choice leads. Each star represents a fork where the person makes their choice.

  1. Write the first step into the box you see in the center of the screen.
  2. Add your first fork by creating two passages. Each time you use [[ ]] it will create a new passage.In my example, these are:

[[The cat decides to take a nap.]]

[[The cat decides to chase the dog.]]

  1. Go into one of these and add the next option. So for my “the cat decides to chase the dog” I will add [[Bad cat!]]
  2. Continue adding your passages until you are satisfied with your narrative. You can loop to other passages by placing them into your text. For example, adding [[Caturday]] to my text will create a link back to the starting passage.
  3. Go to your start page and click “test from here”
  4. Once your story is completed and you’ve tested it, go to Build > Publish to File and download it as an .html file.
  5. Open this file in your browser to make sure it is saved properly.
  6. If it did, then upload this file to the assignment on Canvas.
