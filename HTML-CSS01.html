HTML and CSS: Exercise 01

Directions

This page shows an example of very basic HTML with CSS applied to it. For this exercise, you will create:

  1. New text for your main document
  2. New p id's (6)
  3. New CSS for these p id's
  4. New text in your table
  5. New CSS for your table
The Helpful Tools on Canvas, including W3Schools and MDN, can be of assistance in knowing what the selectors, properties, and values mean. There are also tools for selecting your colors.
