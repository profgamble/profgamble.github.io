Creating an HTML Book

An HTML book is created by linking together multiple HTML files in a chapter format. You can see an example of HTML books on Project Gutenberg.

In this exercise, we will be working with both internal and external links to create two different HTML books. Internal links are within the same document, while external links lead to other documents. You can learn more about the difference between these links by followng this external link to the exercise directions.

For our activity, we will be creating both an internally-linked HTML book and an externally-linked HTML book.

For the externally-linked book, we will be using the following files:

  • Chapter I, The House at Pooh Corner, abridged chapter
  • Chapter II, The House at Pooh Corner, abridged chapter
  • Chapter III, The House at Pooh Corner, abridged chapter

You will start an HTML document. Write a brief introduction, then link to the files.

  • Chapter I, The House at Pooh Corner, abridged
  • Chapter II, The House at Pooh Corner, abridged
  • Chapter III, The House at Pooh Corner, abridged

You have now completed an externally linked book. Save the .html file with using the file convention LastName_ExternalBook.html

Now you will create an internally linked HTML book. This is the main format for HTML books, one that organizations like Project Gutenberg use to create their books.

The process of creating an internally linked HTML book is a bit more complex than an externally linked HTML book. You will need to define ids within the document that allow for the links to go to specific places.

Start with a new .html file. Copy and paste the text from Pupper Ipsum into your file.

At the h2 for Chapter One: Puppers, place an id labeling it as "chapter1". Do the same for h2, labeling it as "chapter2". Return to your list of chapters and place links to #chapter1 and #chapter2.

Now we wil include images in the text.

Inside of your internally linked book, insert these two images into your document with alternative (alt) text. Include the dog with the puppers chapter and the cat with the meowers chapter. For now, do not worry about the size of the image.

  • https://profgamble.github.io/winniedog.jpg
  • https://profgamble.github.io/winniecat.jpg

You have now completed an internally linked book with images. Save the .html file with using the file convention LastName_InternalBook.html

Open both books in your browser window and fix any errors. Then submit both of them.
